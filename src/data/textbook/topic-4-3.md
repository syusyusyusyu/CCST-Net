## 概要

ネットワークデバイス（ルーター、スイッチ、ファイアウォールなど）には、さまざまな種類のポート（接続口）が搭載されています。それぞれのポートは異なる目的で使用され、対応するケーブルやコネクタも異なります。CCST 試験では、各ポートの種類、用途、および物理的な特徴を理解していることが求められます。

## コンソールポート

<KeyTerm term="Console Port" reading="コンソールポート">ネットワークデバイスに直接接続して CLI（コマンドラインインターフェース）にアクセスするための管理用ポートです。ネットワーク接続がなくてもデバイスを設定できます。</KeyTerm>

コンソールポートは、デバイスの初期設定やトラブルシューティング時に最も重要なポートです。以下の特徴があります。

- **RJ-45 コンソールポート**: 従来のコンソールポートで、水色のラベルが付いています。ロールオーバーケーブル（コンソールケーブル）で PC のシリアルポートに接続します。
- **USB コンソールポート**: 新しい Cisco デバイスには USB Type Mini-B または USB Type-C のコンソールポートが搭載されています。PC の USB ポートに直接接続できるため、シリアルポートのない最近の PC でも利用可能です。

コンソール接続のデフォルト設定は、ボーレート 9600 bps、データビット 8、パリティなし、ストップビット 1、フロー制御なし（9600 8N1）です。

<KeyPoint>
- コンソールポートはデバイスの初期設定・トラブルシューティングに使用する最も重要な管理ポート
- RJ-45（水色ラベル）と USB の 2 種類がある
- ネットワーク接続が不要なため、デバイスにアクセスできなくなった際の最後の手段となる
</KeyPoint>

## シリアルポート

<KeyTerm term="Serial Port" reading="シリアルポート">WAN 接続に使用されるポートです。T1/E1 回線や ISDN などの従来の WAN 技術で使われます。シリアル通信でデータを 1 ビットずつ順番に送信します。</KeyTerm>

シリアルポートは主に WAN リンクの接続に使われます。ルーターに搭載されていることが多く、DB-60 や Smart Serial などのコネクタが使用されます。

シリアル接続では、DCE（Data Communications Equipment）と DTE（Data Terminal Equipment）の区別があります。DCE 側がクロック信号を提供し、通信速度を制御します。ISP 側のデバイスが DCE、ユーザー側のルーターが DTE となるのが一般的です。

現在はイーサネットベースの WAN 接続が主流となっているため、シリアルポートの使用頻度は減少していますが、試験の出題範囲として理解しておく必要があります。

## イーサネットポート

<KeyTerm term="Ethernet Port" reading="イーサネットポート">イーサネットネットワークに接続するための標準的なポートです。RJ-45 コネクタを使用し、ツイストペアケーブルで接続します。</KeyTerm>

イーサネットポートは、ネットワークデバイスで最も一般的に使用されるポートです。速度によって以下の種類があります。

| ポートの種類 | 速度 | 表記例 |
|---|---|---|
| FastEthernet | 100 Mbps | Fa0/1, Fe0/1 |
| GigabitEthernet | 1 Gbps | Gi0/1, Ge0/1 |
| TenGigabitEthernet | 10 Gbps | Te0/1 |

スイッチには多数のイーサネットポートが搭載されており、PC やサーバー、他のスイッチなどを接続します。ルーターにもイーサネットポートがあり、LAN への接続や他のルーターとの接続に使用されます。

## 光ファイバーポートと SFP モジュール

<KeyTerm term="SFP" reading="エスエフピー">Small Form-factor Pluggable の略。ネットワークデバイスに挿入する小型のトランシーバーモジュールです。銅線（RJ-45）や光ファイバーなど、さまざまな接続メディアに対応するモジュールを挿し替えて使用できます。</KeyTerm>

SFP ポートは、モジュール式のインターフェースポートです。必要に応じて異なる種類の SFP モジュールを挿入することで、さまざまなケーブルタイプや速度に対応できます。

**主な SFP モジュールの種類:**

| モジュール | 速度 | 接続メディア |
|---|---|---|
| SFP | 最大 1 Gbps | 光ファイバーまたは銅線 |
| SFP+ | 最大 10 Gbps | 光ファイバーまたは銅線 |
| QSFP+ | 最大 40 Gbps | 光ファイバー |
| QSFP28 | 最大 100 Gbps | 光ファイバー |

光ファイバー用の SFP モジュールには、シングルモード（長距離）とマルチモード（短距離）の 2 種類があります。光ファイバーケーブルのコネクタには LC コネクタや SC コネクタが使用されます。

<KeyPoint>
- SFP はモジュール式のため、同じポートで異なるケーブルタイプに対応可能
- SFP（1G）、SFP+（10G）、QSFP+（40G）、QSFP28（100G）と速度に応じたモジュールがある
- 光ファイバー SFP にはシングルモード（長距離）とマルチモード（短距離）がある
</KeyPoint>

## USB ポート

Cisco デバイスの USB ポートは、主に以下の目的で使用されます。

- **USB コンソール接続**: 前述のとおり、コンソールアクセスに使用します
- **USB フラッシュドライブ**: IOS イメージのバックアップや復元、設定ファイルのコピーに使用します
- **外部ストレージ**: ログファイルの保存などに使用できます

USB ポートは一般的に USB Type-A（フラッシュドライブ用）と USB Mini-B/Type-C（コンソール用）の 2 種類が搭載されています。

## PoE 対応ポート

<KeyTerm term="Power over Ethernet" reading="パワーオーバーイーサネット">イーサネットケーブル（Cat5e 以上）を通じて、データ通信と同時に電力を供給する技術です。別途電源ケーブルを配線する必要がないため、IP 電話やアクセスポイントの設置が容易になります。</KeyTerm>

PoE 対応スイッチのポートは、接続されたデバイスに電力を供給できます。PoE を使用する主なデバイスには以下があります。

- IP 電話
- ワイヤレスアクセスポイント
- セキュリティカメラ（IP カメラ）
- IoT デバイス

**PoE の規格:**

| 規格 | 供給電力（最大） | ケーブル要件 |
|---|---|---|
| PoE (802.3af) | 15.4 W | Cat5e 以上 |
| PoE+ (802.3at) | 30 W | Cat5e 以上 |
| PoE++ (802.3bt) | 60 W / 100 W | Cat5e 以上（Cat6a 推奨） |

PoE 対応スイッチには電力バジェット（供給可能な総電力量）があり、接続するデバイスの合計消費電力がこの上限を超えないように注意する必要があります。

<KeyPoint>
- PoE により、イーサネットケーブル 1 本でデータ通信と電力供給が可能
- IP 電話、アクセスポイント、セキュリティカメラなどで広く利用される
- PoE（15.4W）、PoE+（30W）、PoE++（60W/100W）の 3 つの規格がある
</KeyPoint>

## ポートの識別と命名規則

Cisco デバイスのポートには、以下のような命名規則があります。

```
インターフェースタイプ スロット番号/ポート番号
```

例:
- `GigabitEthernet0/1`（Gi0/1）: スロット 0 の GigabitEthernet ポート 1
- `FastEthernet0/24`（Fa0/24）: スロット 0 の FastEthernet ポート 24
- `Serial0/0/0`（Se0/0/0）: シリアルポート

モジュラー型のルーターやスイッチでは、スロット番号が追加されて `GigabitEthernet0/0/0` のような 3 段階の番号体系になることもあります。

## まとめ

- コンソールポート（RJ-45 水色 / USB）はデバイスの直接管理に使用する必須ポートです
- イーサネットポートは RJ-45 コネクタを使用し、FastEthernet（100M）、GigabitEthernet（1G）、TenGigabitEthernet（10G）があります
- SFP モジュールは挿し替え可能なトランシーバーで、光ファイバーや銅線に対応します
- PoE 対応ポートはイーサネットケーブル経由で電力を供給でき、IP 電話やアクセスポイントで活用されます
- Cisco デバイスのポートは「インターフェースタイプ + スロット/ポート番号」の命名規則で識別します
